# Mentorak 
Learn Anything with AI-Generated Quizzes


## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Authentication**: Firebase Auth
- **Database**: Firestore
- **AI Services**: OpenAI API (GPT-4)
- **Forms**: React Hook Form
